# My Finance App

Personal finance tracker to monitor income, expenses, and spending patterns in one dashboard.

## MVP Features:

### Auth
- Register / Login / Sign out
- Personal data per user

### Dashboard

- Total income card
- Total expenses card
- Net balance card
- Expense breakdown pie chart by category
- 
### Transactions

- Add transaction (name, category, amount, date, type: income/expense)
- Edit transaction
- Delete transaction
- Search transactions
- Paginated list (10 per page)

Export to CSV
### Categories:
- Food
- Rent
- Entertainment
- Salary
- Other
